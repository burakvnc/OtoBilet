
# Proje Bilgisi

Otobüs Bileti Alma Uygulaması


## Kullanılan Diller

**CLI** Expo@48.0.18

**Frontend:** React Native@0.71.8

**Backend:** Firebase@9.23.0


## Yükleme 

Yüklemek için gerekenler

```bash 
  npm i
  npm start
```
